<h2>📌 Project Overview</h2>

<p>
<b>ClearAir AI</b> is a smart air quality monitoring web application designed to help users understand pollution levels and environmental conditions in real time.
</p>

<p>
The platform presents air quality information through an interactive dashboard, allowing users to easily track pollution trends and environmental health indicators.
</p>

<p>
The goal of this project is to increase awareness about pollution and provide a simple tool for monitoring air quality.
</p>

<hr>

<h2>✨ Features</h2>

<ul>
<li>🌍 Real-time Air Quality Monitoring</li>
<li>📊 Interactive Data Visualization</li>
<li>⚡ High performance frontend powered by Vite</li>
<li>🎨 Modern UI built using Tailwind CSS</li>
<li>📱 Fully responsive design</li>
<li>🧠 AI-ready architecture for future predictive analysis</li>
</ul>

<h2>⚙️ Installation</h2>

<p><b>Clone the repository</b></p>

<pre>
git clone https://github.com/siddhant210/Air-Quality.git
</pre>

<p><b>Navigate to the project directory</b></p>

<pre>
cd Air-Quality
</pre>

<p><b>Install dependencies</b></p>

<pre>
npm install
</pre>

<p><b>Run development server</b></p>

<pre>
npm run dev
</pre>

<p>The project will run on:</p>

<pre>
http://localhost:5173
</pre>
